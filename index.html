📲 Instalar no Celular
Instale o Vertomaker como um app no seu celular ou computador. Ele abre em tela cheia, fica no menu de apps e funciona sem precisar abrir o navegador.
💡 Esse botão só aparece se o app estiver publicado num endereço https:// (ex.: GitHub Pages) com os arquivos sw.js e manifest.webmanifest na mesma pasta do HTML. Abrindo o arquivo direto do celular, sem hospedar, o app funciona normalmente — só essa instalação automática não fica disponível.
💾 Backup e Restauração
Todos os seus dados (clientes, tarefas, estoque, financeiro, etc.) ficam salvos neste dispositivo. Exporte um arquivo de backup periodicamente e guarde no Google Drive, e-mail ou outro lugar seguro — assim você pode restaurar tudo caso troque de celular ou apague o navegador.
💡 Dica: depois de baixar o arquivo de backup, envie-o para o Google Drive (app Drive → "+" → Upload) para manter uma cópia segura na nuvem.
🗑️ Dados
Limpar todos os dados salvos (tarefas, transações, etc.). Esta ação não pode ser desfeita.
